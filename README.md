
# Metanoia v1.0

Metanoia is a cross-platform mobile application built for 
stress relief and personal well-being. It offers variety of features like
meditation, calming sounds, games, etc.



## Features

- Chabot Milo
- Zen Mode
- Sleep sounds
- Bubble Pop game
- Meditation 
- Workout videos
